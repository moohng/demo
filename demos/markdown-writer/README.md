# Markdown 公众号写作工具

一个功能强大的在线 Markdown 编辑器，专门用于生成公众号文章模板。

## 功能特性

- 📝 **实时 Markdown 编辑** - 支持标准的 Markdown 语法
- 🎨 **多种模板主题** - 内置 4 种不同风格的公众号模板
- 🎯 **自定义主题** - 可自由调整颜色和样式
- 👀 **实时预览** - 左侧编辑，右侧实时预览效果
- 💾 **自动保存** - 内容自动保存到本地存储
- 📤 **一键导出** - 导出为完整的 HTML 文件
- 🌙 **暗色模式** - 支持亮色/暗色主题切换
- ⌨️ **键盘快捷键** - 支持常用快捷键操作

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 使用说明

### 基本操作

1. **编辑内容**：在左侧编辑器中输入 Markdown 内容
2. **实时预览**：右侧会实时显示公众号样式效果
3. **选择模板**：通过下拉菜单选择不同的模板风格
4. **自定义颜色**：在下方面板中调整主题颜色
5. **导出文章**：点击右上角按钮导出为 HTML 文件

### 支持的 Markdown 语法

- 标题：`# H1`、`## H2`、`### H3`
- 粗体：`**粗体**`
- 斜体：`*斜体*`
- 列表：`- 项目`、`1. 项目`
- 代码块：\`\`\`javascript\ncode\n\`\`\`
- 引用：`> 引用内容`
- 链接：`[文本](URL)`
- 图片：`![描述](图片URL)`

### 键盘快捷键

- `Ctrl + S` - 导出文章
- `Ctrl + D` - 切换主题

## 模板说明

### 默认模板
- 经典绿色主题，适合大多数公众号

### 优雅风格
- 红色主题，适合文艺类内容

### 现代风格
- 蓝色主题，适合科技类内容

### 经典风格
- 棕色主题，适合传统类内容

## 技术栈

- **构建工具**：Vite
- **Markdown 解析**：marked
- **代码高亮**：highlight.js
- **样式**：纯 CSS3
- **类型检查**：TypeScript
